Class LilyBlog::Post
In: lib/lilyblog/post.rb
Parent: Object

Main post class for LilyBlog

The initialize method for Post requires a filename to load. Post files are a section of yaml, followed by a blank line and then a post formatted with Markdown. Please see the README file for more information about how to format posts that this class uses.
