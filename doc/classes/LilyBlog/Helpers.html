Public Instance methods

class_for_tag(tag)

Returns the class name to use on a tag link

css_attrs(sheet, media = 'all')

Like the sinatra helper for html_attrs, this returns the css tag attributes for a given style sheet.

For example, to load the /css/layout.css file use: %link{ css_attrs(:layout) }

feed_url()

This returns the feed url for the current page

feed_urls()

This returns an array of [feed_name, feed_url] for any feeds applicable to the current page.

max_page()

The maximum page needed to show all posts

I would like to put this in the model, but it needs to know if we‘re on a tag page or not

next_page_url()

Adds the page parameter onto the current url for the next page

page_url(page)

Adds the page parameter onto the current url for a given page

partial(name, obj, locals = {})

Very simple partial function

Takes a template name and an object to load as the main object (named the same as the template name). Then takes a hash of other local variables.

Example: partial :post, @post, :comments => true

post_url(post)

Given a post model, this returns the url of the post (for linking)

prev_page_url()

Adds the page parameter onto the current url for the previous page

search_url(search)

Returns the search url for a given term.

tag_url(tag)

Returns the tag url for a given tag

title()

Returns the title that should be on this page
